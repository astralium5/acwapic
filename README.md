# ACWAPIC

ACWAPIC (Astral's CatWeb API Client, or just CWAPIC, /æk.wɑː.pɪk/) is a Python Utility to easily create APIs for the Roblox game "CatWeb".

## How??

In CatWeb, there exists a block-based script element that can do many things. The one we're looking at here is the deceptively simple "Log" block, which simply logs anything you type into the console. **Yes, that developer console.**
Using some magic, We can pipe these logs into a custom function to do, well, everything really. It's up to you to figure out what to do with this.
Guess what CatWeb also has? Broadcasts. Which broadcast to everyone on that site. We can use this to our advantage along with input fields to send and return data with each-other. This also used to be how you made U2U communication on CatWeb before Roblox decided "Screw you. 12 presets only. No hi hello how are you too.". Good job Roblox...
